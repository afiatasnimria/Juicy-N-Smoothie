# Huicy & Smoothy

A modern juice and smoothie ordering platform built for fast, fresh, and healthy beverage delivery.  
Huicy & Smoothy allows users to browse drinks, customize ingredients, place orders, and manage deliveries with a smooth user experience.

---

## ✨ Features

- 🍓 Browse smoothies, juices, and healthy drinks
- 🛒 Add products to cart
- 💳 Secure checkout system
- 📱 Fully responsive design
- 🔍 Search and filter drinks
- ❤️ Favorite / wishlist support
- 🧾 Order history
- 👨‍💼 Admin dashboard for product management
- 🚚 Delivery tracking support

---

## 🛠️ Tech Stack

### Frontend
- HTML / CSS / JavaScript
- React.js / Next.js *(optional)*
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB / MySQL
